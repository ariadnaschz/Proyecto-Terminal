# Proyecto-Terminal

[![Abrir en GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/ariadnaschz/Proyecto-Terminal?quickstart=1)

Simulación del modelo de Simon Mukwembi y cálculo del número VIH(G) en distintas familias de gráficas.

Este repositorio contiene la implementación computacional utilizada en el proyecto terminal “Dinámica de la infección por VIH en los ganglios linfáticos: un enfoque de teoría de gráficas”.

El objetivo principal es simular el modelo propuesto por Simon Mukwembi para estudiar la propagación del VIH en estructuras de tipo grafo y calcular el número VIH(G), definido como el mínimo valor del parámetro de reemplazo R para el cual toda configuración inicial conduce a la curación del sistema.

El código está desarrollado en Python/SageMath 9.8 e incluye:

* La simulación de las reglas R1, R2 y R3 en pasos de tiempo discretos,

* La generación de familias de gráficas (Pn, Cn, Kn, Wn, Sn, Kn,m, Rn,m),

* El cálculo experimental del número VIH(G),

* Herramientas para visualizar la dinámica temporal de la infección,

* Tablas y resultados incluidos en el proyecto.

Este repositorio busca ser una herramienta reproducible, abierta y útil para estudiantes e investigadores interesados en dinámica en redes, teoría de gráficas y modelos matemáticos de propagación.
